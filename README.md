You must have Java installed!
````
java -version
````

Download Clojure
````
wget http://central.maven.org/maven2/org/clojure/clojure/1.6.0/clojure-1.6.0.jar
````

Run Clojure REPL
````
java -jar clojure-1.6.0.jar
````

Run a script
````
java -jar clojure-1.6.0.jar hello.clj
````
